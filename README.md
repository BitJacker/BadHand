# Bad Hand v1.0 - Cybersecurity Tools

A collection of security testing tools to test common vulnerabilities in networks and web applications. Created by BitJacker.

## Included Tools:

1. **DDOS (udp_attak.py)**: A tool to launch UDP-based DDoS attacks.
2. **DEAUTHER (deauther.py)**: A tool to disconnect devices from a Wi-Fi network.
3. **SCAN (scan.py)**: A scanner to find devices on a network.
4. **XSS SCANNER (xss_scanner.py)**: A scanner for Cross-Site Scripting (XSS) vulnerabilities.
5. **SQL INJECTION SCANNER (sqli_scanner.py)**: A scanner for SQL Injection vulnerabilities.
6. **DIRECTORY BRUTEFORCE (dir_bruteforce.py)**: A tool for directory brute force attacks.
7. **SSL/TLS CERTIFICATE CHECKER (ssl_checker.py)**: A tool to check SSL certificate validity of a domain.
8. **RCE TEST (rce_test.py)**: A tool for testing Remote Code Execution (RCE) on a URL.
9. **CREDENTIAL STUFFING (credential_stuffing.py)**: A tool for credential stuffing attacks.
10. **SUBDOMAIN ENUMERATION (subdomain_enum.py)**: A tool for enumerating subdomains of a domain.

## Requirements

1. Python 3.x
2. Python dependencies (install them via `pip` if not already installed).

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/BitJacker/bad-hand.git
   cd bad-hand
   
2. Install required dependencies (if present):
   pip install -r requirements.txt

Run the main script:  python3 main.py

Select one of the tools from the menu to get started.

Note: Some tools may require administrator privileges (e.g., deauther). Use sudo if needed
